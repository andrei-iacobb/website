# iacob.uk

Personal domain website for Andrei Iacob, built with Next.js 15.

## Features

- Modern authentication system with device recognition
- Beautiful gradient design with purple-to-pink theme
- Responsive design optimized for all devices
- Custom components built with Tailwind CSS and shadcn/ui

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js 15 App Router pages and layouts
- `components/` - React components including UI components
- `public/` - Static assets including custom logos and icons

## Built With

- [Next.js 15](https://nextjs.org) - React framework
- [Tailwind CSS](https://tailwindcss.com) - Styling
- [shadcn/ui](https://ui.shadcn.com) - UI components
- [Lucide React](https://lucide.dev) - Icons
- Custom authentication system
